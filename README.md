# Hector el Sysadmin

![hector](./img/hector.png)

* Encargado de:
  * Anuncios públicos.
  * Entregar información del server.
  * Gatillar copias de seguridad.

## Uso

Incluir un archivo `.env` en el directorio donde se ejecuta el programa:

```
bot_token="<token>"
channel_id="<chat-id>"
api_url="<url>"
```
